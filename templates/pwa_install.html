Comment installer l'application

Android (Chrome/Edge)

  1. Ouvrez cette page dans Chrome ou Edge sur votre téléphone
  2. Appuyez sur le bouton "Installer l'application" qui apparaît
  3. Ou utilisez le menu ⋮ → "Installer l'application" ou "Ajouter à l'écran d'accueil"
  4. Confirmez l'installation
  5. L'icône BENJ INSIDE apparaîtra sur votre écran d'accueil

iPhone/iPad (Safari)

  1. Ouvrez cette page dans Safari sur votre iPhone/iPad
  2. Appuyez sur le bouton de partage en bas de l'écran
  3. Faites défiler et sélectionnez "Sur l'écran d'accueil"
  4. Modifiez le nom si souhaité et appuyez sur "Ajouter"
  5. L'icône BENJ INSIDE apparaîtra sur votre écran d'accueil

Ordinateur (Chrome/Edge/Firefox)

  1. Ouvrez cette page dans Chrome, Edge ou Firefox
  2. Cliquez sur l'icône d'installation dans la barre d'adresse
  3. Ou utilisez le menu → "Installer BENJ INSIDE"
  4. Confirmez l'installation
  5. L'application sera accessible depuis votre bureau ou menu démarrer

Le bouton d'installation apparaîtra automatiquement si votre navigateur le supporte
